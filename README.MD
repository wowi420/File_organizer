## File Organizer
A program that organizer files in one specific Directories without
bleeding into other Dir

## How To
1.make sure you know the exact name of your folder
2.run the script 
# show
>>>What dir you wanna organize:[DIRECTORIES/FOLDER]
>>>The process is completed

## What file it will handle?
.png
.jpg
.jpeg
.docx
.pptx
.txt

## Preview of what it would be
your-folder/

│   ├── .png

└── documents/
    ├── .docx
